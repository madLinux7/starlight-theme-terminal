## What this theme offers

<FeatureGrid columns={2}>
  <Card title="FeatureGrid" icon="seti:csv">
    Showcase your project's most relevant features using the `FeatureGrid` component shipped in this theme - used right here!
  </Card>
  <Card title="Zero-config setup" icon="rocket">
    Drop it into `astro.config.mjs` and ship. No CSS overrides or token hunting required.
  </Card>
  <Card title="Dark and light parity" icon="moon">
    Both modes are tuned for readability. Code blocks, callouts, and diagrams stay legible in either.
  </Card>
  <Card title="Starlight-native" icon="star">
    Built on Starlight primitives. Every component, shortcode, and plugin keeps working.
  </Card>
</FeatureGrid>
